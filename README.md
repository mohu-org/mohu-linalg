# mohu-linalg

Pure-Rust linear algebra routines — usable standalone or as the linalg backend for [mohu](https://github.com/mohu-org/mohu).

## Routines

- Matrix multiplication (matmul)
- LU decomposition
- QR decomposition
- SVD
- Cholesky decomposition
- Norms (L1, L2, Frobenius, nuclear)
- Linear solvers
- Eigenvalues / eigenvectors
